O desafio propõe a criação de um caderno temático no NotebookLM, reunindo de três a cinco fontes abertas em texto ou PDF sobre um assunto financeiro introdutório. 

Tema: Métodos e Estratégias para o Controle de Gastos Pessoais

1. Introdução e Objetivo
Este projeto consistiu na criação de um ambiente de aprendizagem ativa utilizando a ferramenta NotebookLM. O objetivo central foi transformar informações dispersas sobre educação financeira em um guia prático e estruturado. O foco recaiu sobre a metodologia de controle de gastos, buscando não apenas entender "o que" fazer, mas "como" implementar estratégias eficientes de organização financeira pessoal.

2. Curadoria de Fontes
Foram selecionadas 5 fontes de alta relevância. A escolha das fontes visou cobrir três pilares:

Fundamentos teóricos da organização financeira.
Ferramentas práticas de registro de despesas.
Estratégias comportamentais para manutenção do orçamento.

3. Desenvolvimento e Metodologia de IA
Utilizando o NotebookLM, o material foi processado para gerar uma base de conhecimento privada. O processo de estudo foi conduzido através de:

Perguntas Estratégicas: Questionamentos direcionados para identificar os métodos mais eficazes citados nas fontes (ex: Regra 50/30/20, kakebo).
Engenharia de Prompts: Foram testadas variações de comandos para extrair resumos comparativos e planos de ação, garantindo que as respostas estivessem estritamente fundamentadas nas fontes carregadas.
Validação de Referências: Cada insight gerado pela IA foi conferido com as citações diretas dos documentos originais.
